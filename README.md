# BioApi - Система автоматического обновления товаров

## 📋 Описание

Система автоматического сбора данных о товарах из API Bio и обновления остатков и цен в Bitrix CMS.

## 🚀 Основные компоненты

### 1. `bio_api.py` - Основной Flask сервер
- **Функции:**
  - Сбор данных о товарах из API Bio
  - Расчет цен по формуле: `(X/1.2*курс*1.12+доставка)*1.18`
  - Сохранение данных в SQLite базу
  - Автоматическое обновление в 5:00 утра (время Франкфурта)
  - Запуск скрипта обновления остатков после сбора данных

### 2. `update_stocks_bio.py` - Скрипт обновления остатков
- **Функции:**
  - Чтение данных из SQLite базы
  - Округление цен до 100 вверх
  - Отправка данных на сервер Bitrix

### 3. `valute.py` - Обновление курсов валют
- **Функции:**
  - Парсинг курсов валют с сайта mig.kz
  - Сохранение курсов в `info.py`

### 4. `data_stocks_bio.php` - PHP скрипт на сервере
- **Функции:**
  - Обновление остатков и цен товаров в Bitrix
  - Установка статусов наличия

## ⏰ Расписание работы

- **Время обновления:** 5:00 утра (время Франкфурта, UTC+1)
- **Частота:** Ежедневно
- **Процесс:**
  1. Обновление курсов валют
  2. Сбор данных из API Bio
  3. Расчет цен и сохранение в базу
  4. Запуск скрипта обновления остатков
  5. Ожидание до следующего дня

## 📊 Формула расчета цен

```
Цена = (Исходная_цена / 1.2 * курс_валюты * 1.12 + стоимость_доставки) * 1.18
```

### Стоимость доставки:
- **До 30 кг:** 31,900 тг
- **30-300 кг:** 7500 + (вес-30)*179 + 10000 + (вес-30)*20 + 10400 + 4000 + (вес-30)*15
- **300-1000 кг:** Сложная формула с компонентами
- **Свыше 1000 кг:** Используется формула для 1000 кг

### Округление цен:
Все цены округляются до 100 вверх (13480 → 13500)
